## 📖 Project Description

The **Student Management System** is a Java-based console application that helps manage student records efficiently. This project allows users to add, remove, search, and display student information through a simple menu-driven interface.

It demonstrates the use of **Object-Oriented Programming (OOP)** concepts and Java Collections Framework.

---

## ✨ Features

✅ Add Student Records
✅ Remove Student Records
✅ Search Student by Roll Number
✅ Display All Student Records
✅ Menu-Driven Interface
✅ User-Friendly Console Application
